# building-games
Building Games

Et JavaScript-spil bygget som del af et kort kursusforløb (2,5 ECTS) på Roskilde Universitet.

Om projektet

Opgaven gik ud på at bygge et fungerende spil i JavaScript. GitHub Copilot var tilladt som hjælpeværktøj undervejs i udviklingen.

Teknologier


JavaScript


Læring

Spillogik, brugerinput og grundlæggende programmeringsstrukturer i JavaScript — samt erfaring med at bruge AI-værktøjer som en del af udviklingsprocessen.
